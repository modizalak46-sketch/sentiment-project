# Sentiment- Project

## Description
This project performs Sentiment Analysis. Users can input text and get sentiment predictions (Positive, Negative, Neutral). The backend is written in Python (app.py) and frontend uses HTML templates.

## Project Structure
Sentiment-Project/
├── templates/
│   └── index.html
├── app.py
├── requirements.txt
└── README.md

## Features
- Web interface for entering text  
- Predicts sentiment using Python backend  
- Easy to run and extend  

## Installation
1. Clone or download the repository  
2. Create a virtual environment (optional)
3. Install dependencies
4.  Run the application

## Usage
- Open the web application in your browser  
- Enter text in the input box  
- View the sentiment prediction
